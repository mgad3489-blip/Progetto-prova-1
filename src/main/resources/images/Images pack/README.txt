Fantasy Tower Defense Starter Pack

This package is structured for itch.io asset distribution.

Current package contents:
- 30 curated PNG assets exported from the preview sheet
- 4 sprite sheets in `Sprite-Sheets/`
- Product text, manifest, and licensing docs
- Build script for versioned release zips

Current status:
- Assets are grouped by actual content instead of generic placeholder folders.
- This is still a starter pack, but the exported files now use curated names and more accurate categories.

Quality goals:
- Clean folder organization
- Consistent naming
- Clear preview and documentation
- Versioned release notes and metadata

Important legal note:
- Do not redistribute third-party assets unless license terms explicitly allow it.
- Keep `Tiny Swords.zip` only as private reference; it is excluded from the release build.

Unity import suggestions:
- Texture Type: Sprite (2D and UI)
- Filter Mode: Bilinear or Point (based on style)
- Compression: Low or None
- Keep pixels-per-unit consistent across all files
