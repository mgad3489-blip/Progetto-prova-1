# Pack Quality Checklist

Use this before every itch.io upload.

## Content quality
- [ ] All included files are your own work or licensed for redistribution.
- [ ] No third-party zip files are included.
- [ ] Assets are separated by category and easy to navigate.
- [ ] File names are consistent and descriptive.

## Technical quality
- [ ] PNGs use transparent background where needed.
- [ ] Resolution/style is consistent.
- [ ] No duplicate junk files (`.DS_Store`, temporary exports).
- [ ] README clearly states what is included and what is not.

## Store quality
- [ ] At least 3 screenshots on the itch page.
- [ ] A cover image and one collage preview are prepared.
- [ ] Description includes engine compatibility and usage terms.
- [ ] AI-assisted disclosure is present if applicable.

## Packaging quality
- [ ] Build script runs cleanly.
- [ ] Zip name is versioned.
- [ ] Release notes updated for current version.
- [ ] Manifest and metadata included in Docs.
