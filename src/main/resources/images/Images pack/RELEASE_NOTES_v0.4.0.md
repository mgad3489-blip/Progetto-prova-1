# Release Notes - v0.4.0

Release date: 2026-03-12

## Fixed
- Replaced shape-based auto classification with a curated 30-asset export map.
- Removed misleading duplicate crops and inaccurate folder placement.
- Renamed assets to content-based filenames.

## Added
- 4 sprite sheets for browsing and import.
- More accurate structure for heroes, enemies, UI, props, paths, resources, and towers.
- Improved manifest descriptions for each exported asset.

## Notes
- Assets in this release are still extracted from the preview source image.
- This version is meant to be cleaner and more believable as a starter pack.
